## О проекте

Этот проект представляет собой **файловый менеджер с графическим интерфейсом**, разработанный на Python с использованием фреймворка Flet. Приложение позволяет выполнять основные операции с файлами и папками через интуитивно понятный интерфейс.

### Особенности проекта:
- ✅ Работает на Windows, Linux, macOS
- ✅ Может запускаться как десктопное или веб-приложение
- ✅ Поддержка русского языка
- ✅ Подтверждение опасных операций (удаление)
- ✅ Всплывающие подсказки для всех кнопок

## Возможности

### Основные операции
| Операция | Описание | Где работает |
|----------|----------|--------------|
| 📁 Выбор файла | Выбор одного файла через системный диалог | Файлы |
| 📂 Выбор папки | Выбор папки через системный диалог | Папки |
| 📋 Копирование | Копирование файла с префиксом "copy_" | Только файлы |
| 🗑️ Удаление | Удаление файла или папки (с подтверждением) | Файлы и папки |
| 🔢 Подсчет | Подсчет количества файлов в папке | Только папки |
| 🔍 Поиск | Поиск файлов по шаблону (регулярное выражение) | Только папки |
| 📊 Анализ | Анализ структуры папки с размерами файлов | Только папки |

### Особенности операций

#### Копирование
- Создает копию файла в той же папке
- Имя копии: `copy_оригинальное_имя`
- Пример: `document.txt` → `copy_document.txt`

#### Удаление
- Выводит диалоговое окно подтверждения
- Работает как с файлами, так и с папками
- Папки удаляются рекурсивно (вместе со всем содержимым)

#### Поиск
- Поддерживает шаблоны (wildcards)
- Примеры шаблонов: `*.txt`, `*.py`, `test*`, `*.jpg`
- Показывает первые 5 результатов (если больше - указывает количество)

#### Анализ
- Показывает структуру папки
- Отображает размер каждого файла
- Рекурсивно обходит все вложенные папки
- Вывод ограничен 500 символами для удобства просмотра
