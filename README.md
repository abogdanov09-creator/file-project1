
# Файловый менеджер с графическим интерфейсом

Консольный файловый менеджер из прошлого ДЗ теперь с удобным графическим интерфейсом.

## Возможности

### Операции с файлами:
- **Копирование** - копирует выбранный файл с префиксом "copy_"
- **Удаление** - удаляет файл или папку (с подтверждением)
- **Подсчет файлов** - показывает общее количество файлов в папке
- **Поиск** - ищет файлы по регулярному выражению
- **Анализ** - показывает структуру папки с размерами файлов
