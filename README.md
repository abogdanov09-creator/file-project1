# Файловый менеджер (GUI + CLI)

Консольная утилита и графический интерфейс для управления файлами.

---

## 📦 Установка

```bash
# 1. Клонировать репозиторий
git clone https://github.com/ваш-username/file-project1.git
cd file-project1

# 2. Создать виртуальное окружение (рекомендуется)
python -m venv venv

# 3. Активировать виртуальное окружение
# Windows:
venv\Scripts\activate
# macOS / Linux:
source venv/bin/activate

# 4. Установить зависимости
pip install -r requirements.txt
